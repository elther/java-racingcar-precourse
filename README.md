# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

1. 자동차 클래스 구현
    1) 자동차 이름
    2) 전진 기능
    3) 스탑 기능

2. 일급 콜렉션에 자동차 담기

3. 자동차 이름 입력
    1) (,) 쉼표를 기준으로 분리하기
    2) 자동차 이름 validation
    3) 분리된 자동차 이름의 갯수만큼 자동차 생성

4. 레이싱 게임
    1) 사용자 입력 (turn count)
    2) 자동차별로 random 값 출력 (전진, 스탑)
    3) 순위 출력
    4) 우승자 출력
